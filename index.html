    📘 ヘルプ

    📖 SRC Portalについて

    SRCメンバーのラン＆ウォークやジムへの参加予定を共有し、仲間と楽しく運動を続けるためのアプリです。

    📋 使い方

    1. 現在のユーザーを選択します。
    2. 「ラン＆ウォーク」または「ジム」を選択します。
    3. 参加したい日付をタップします。
    4. 「参加する」または「参加取消」を押します。

    参加人数や参加者はリアルタイムで更新されます。ダッシュボードの「次回イベント」をタップすると、該当月のカレンダーへ移動します。

    📢 お知らせ

    管理者からの正式な連絡です。「お知らせ（○件）」の枠内を上下にスクロールして確認できます。

    💬 みんなの伝言板

    メンバー同士の軽い情報共有に使います。「投稿・一覧」を押し、伝言と掲載期間（3日・7日・14日）を選んで投稿してください。自分の投稿は削除でき、管理者はすべての投稿を削除できます。期限切れの投稿は表示されません。

    📄 利用規約

    SRC Portalをご利用いただきありがとうございます。

    みんなが気持ちよく利用できるよう、以下の内容をご確認ください。

    1. 利用できる人

    SRCメンバーとして管理者が登録した方が利用できます。

    2. 利用について

    • 参加登録・取消はご本人が行ってください。
    • みんなが気持ちよく利用できるよう、ご協力をお願いします。

    3. やってはいけないこと

    • 他人になりすまして利用すること
    • 他人の参加情報を勝手に変更すること
    • アプリの正常な運営を妨げること
    • その他、管理者が不適切と判断する行為

    4. 利用停止について

    次の場合は、管理者の判断で利用を停止（無効化）することがあります。

    • SRCメンバーでなくなった場合
    • 1年以上利用がない場合
    • 利用ルールを守っていただけない場合
    • その他、運営上必要と判断した場合

    再度利用したい場合は、管理者までお気軽にご連絡ください。

    5. アプリについて

    本アプリは、必要に応じて機能の追加・変更・停止を行うことがあります。

    6. 個人情報について

    アプリでは、運営に必要な範囲でメンバー名や参加情報などを管理します。

    取得した情報は、SRC Portalの運営以外の目的には利用しません。

    7. 利用規約の変更

    必要に応じて、本利用規約を変更する場合があります。

    最終更新：2026年7月21日

    🔒 プライバシーポリシー

    SRC Portalでは、運営に必要な範囲で次の情報を取り扱います。

    • メンバー名、管理者設定、招待・登録状況
    • イベントへの参加・取消・当日連絡
    • 登録日時、最終利用日時などの運用情報

    これらの情報は、参加予定の共有、開催判断、メンバー管理およびアプリ改善のために利用し、法令に基づく場合を除いて目的外で第三者へ提供しません。

    端末には、選択したユーザーや招待認証に必要な情報が保存される場合があります。共用端末では利用後の管理に注意してください。

    制定：2026年7月20日
