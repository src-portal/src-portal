SRC Portal Release Notes

Version 1.6.x
- SRC-KYRO-Clubの領土・全国順位・愛知県順位・最新ニュース・次の目標を表示
- 登録メンバー一覧でKYROメンバーに黄色背景・黒文字のKYROバッジを表示
- 管理者がKYROメンバー設定とクラブ情報を更新可能

Version 1.5.x
- 登録メンバーの自己紹介（プロフィール）機能を追加
- 自分のプロフィールを編集し、ほかのメンバーのプロフィールを閲覧可能
- 必須項目と任意項目を分け、ランニングをしないメンバーにも対応

Version 1.4.x
- 翌日の参加予定を知らせるリマインダー機能を追加
- 参加未回答のイベントやジム予定を案内

Version 1.3.x
- 「みんなの伝言板」を追加
- 次回イベントをダッシュボードカードへ集約
- お知らせを件数表示・スクロール対応に整理

Version 1.2.x
- 登録メンバー一覧に今月・先月の参加回数切替を追加
- 多言語表示の翻訳漏れを再点検・修正

Version 1.1.x
- 端末・ブラウザの言語を自動判定
- 日本語・英語・韓国語・中国語の基本操作固定文言に対応

Version 1.0.x — Official Release
- 正式版として公開
- Help、利用規約、プライバシーポリシー、リリースノートを整備
- 招待コードによる初回登録に対応
